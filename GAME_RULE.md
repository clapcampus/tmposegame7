# 게임 규칙 (Game Rules)

## 🎮 게임 제목
**Fruit Catcher (과일 받기 게임)**

## 📝 게임 설명
하늘에서 떨어지는 과일을 바구니에 담아 점수를 획득하는 게임입니다.
플레이어는 바구니를 움직여 폭탄을 피하고 맛있는 과일만 받아야 합니다!

## 🎯 게임 목표
- 제한 시간 동안 폭탄을 피해 최대한 많은 과일을 받아 최고 점수를 기록하세요.
- 폭탄을 받으면 즉시 게임이 종료됩니다.

## 🕹️ 조작 방법 (포즈)
웹캠 앞에서 몸을 움직여 바구니를 조종합니다. 화면은 3개의 레인(왼쪽, 중앙, 오른쪽)으로 나뉩니다.

| 포즈 (Class) | 동작 | 설명 |
|---|---|---|
| **왼쪽 (Left)** | 왼쪽 이동 | 바구니를 왼쪽 레인으로 이동 |
| **정면 (Center)** | 중앙 이동 | 바구니를 중앙 레인으로 이동 |
| **오른쪽 (Right)** | 오른쪽 이동 | 바구니를 오른쪽 레인으로 이동 |

> **만세 (Pose)** 기능 추가 가능 (선택 사항): 모든 과일을 끌어당기는 자석 효과 등

## 🍎 아이템 및 점수

| 아이템 | 이미지 | 점수/효과 |
|---|---|---|
| **사과 (Apple)** | 🍎 | +100점 |
| **바나나 (Banana)** | 🍌 | +200점 |
| **오렌지 (Orange)** | 🍊 | +300점 (속도가 빠름) |
| **폭탄 (Bomb)** | 💣 | **GAME OVER** (즉시 종료) |

## ⚖️ 게임 규칙 상세

1. **시작**: 'Start' 버튼을 누르면 카운트다운 후 게임이 시작됩니다.
2. **진행**:
   - 과일과 폭탄이 무작위 위치(3개 레인 중 하나)에서 떨어집니다.
   - 플레이어는 포즈를 취해 해당 위치로 바구니를 옮깁니다.
3. **콤보 시스템**: 과일을 놓치지 않고 연속으로 5개 받으면 **피버 타임** (점수 2배)!
4. **레벨 시스템**: 점수가 1000점 오를 때마다 레벨 업! (과일 낙하 속도 증가)

## 🎨 UI 구성 계획
- **상단**: 현재 점수, 남은 시간, 현재 레벨
- **중앙**: 게임 화면 (플레이어 바구니 + 떨어지는 아이템)
- **화면 좌측 하단**: 웹캠 피드 (내 포즈 확인용)
- **종료 화면**: 최종 점수 및 '다시 하기' 버튼
